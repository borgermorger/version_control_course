# Подсказки по командной строке

Команда смены директории:
```sh
cd c:\folder_name
```

Команда отоброжения текущей директории для MacOs, Linux:
```sh
pwd 
```

Листинг текущей директории Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```

в Linux, MacOs:
```sh
rm <filename>
```
